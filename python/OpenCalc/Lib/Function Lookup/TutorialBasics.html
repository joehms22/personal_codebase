Layout
Being new to OpenCalc, lots of the interface may seem confusing, the menubar is different, there are interesting tabs filled with gibberish, etc. In a short amount of time you will be flying through the program like a pro! But first you will sometimes see things like this: 
==2*2
4
This shows an input (the top line, preceeded by ==) please don't type the two == signs, this is what the program shows you to differentiate between your input and it's output.

Basic Operations (x / + -)
Switch to the tab labeled Pad (It is on the far left), you can do this by pressing the tab, or by going to Pad > Switch To Pad. You will see two boxes and a button in the tab. In the smaller box (the one at the bottom) type in 2+2, then press submit.
