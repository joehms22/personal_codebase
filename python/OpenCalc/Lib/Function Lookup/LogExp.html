E
When doing natural logs you may run across e or E. OpenCalc provides the variable E not to be confused with e which is a user available variable. E = 2.71828183...

Logs
Logs are entered in the following ways: 
The log base ten of a number.
  log(number)
The log of a number with the given base.
  log(number, base)

Natural Logs
Natural Logs are entered in the following ways: 
Natural Log:
    ln(number)
    natlog(number)
There is no difference between these methods.

Exponents
Exponents may be entered in a variety of ways 
Using a carrot.
==2^3
8.0

Using two times signs.
==2**3
8.0

Using exp().
==exp(2,3)
8.0

Roots
Square roots may be achived by using the sqrt function: 
==sqrt(4)
2.0
Other roots may be achived by using the root function: 
==root(8, 3)
2.0
Or by raising the number to 1/root see the Exponents section on how to raise a number. 
==8^(1/3)
2.0
